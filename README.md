## 2. 기본 문법 - 태그

### 줄바꿈과 단락

p 태그를 통해서 단락의 경계를 분명히 하면서
CSS를 통해서 p 태그의 디자인을 자유롭게 변경할 수 있기 때문에
br 태그 보다 p 태그가 더 좋은 선택일 수 있다.
